# Speech Processing Projects

A collection of projects covering core topics in speech and audio signal processing using Python.

## Overview

This repository contains hands-on implementations of various speech processing techniques, ranging from basic audio analysis to feature extraction, speech recognition, and enhancement methods.

## Projects Included

| Project | Focus Area | Format |
|---------|------------|--------|
| Introduction to Speech Processing | Audio loading, waveform visualization, time-domain analysis | Jupyter Notebook |
| Framing, Windowing & Feature Extraction | Short-term energy, Zero Crossing Rate, basic signal framing | Jupyter Notebook |
| Real-Time Speech Recognition | Live transcription using Google Speech API and PocketSphinx | Python + Streamlit |
| Spectral Analysis | FFT, spectrograms, pitch and formant estimation | Jupyter Notebook |
| MFCC Feature Extraction | Mel-frequency cepstral coefficients, pre-emphasis, filter banks | Jupyter Notebook |
| Speech Classification | Feature extraction and basic classification models | Jupyter Notebook |
| Speech Enhancement | Noise reduction using spectral subtraction | Jupyter Notebook |
| End-to-End Speech Pipeline | Audio preprocessing, feature extraction, and analysis | Jupyter Notebook |
| Speech Recognition Experiments | End-to-end feature extraction and model evaluation | Jupyter Notebook |
| Speech-Based Directory System | Interactive speech-driven name lookup system | Python Script |

## Tech Stack

- Python
- Jupyter Notebook / Google Colab
- Librosa, NumPy, Matplotlib, Pandas
- SpeechRecognition, PocketSphinx
- Streamlit

## Key Skills Demonstrated

- Audio signal processing and visualization
- Time and frequency domain analysis
- Feature extraction (MFCC, ZCR, Energy)
- Speech recognition and classification
- Basic speech enhancement techniques

## How to Use

Each project is self-contained. Open the respective `.ipynb` or `.py` file to explore the code and results.
