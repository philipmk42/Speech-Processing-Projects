# 🗣️ Speech-Processing-pgms  
A collection of Python and Jupyter Notebook programs for **Speech Processing Laboratory** exercises.  
This repository covers audio handling, feature extraction, spectral analysis, speech enhancement, and real-time speech recognition applications.

---

## 📘 Lab Program Descriptions

### 🔹 **Lab 1 — Introduction to Speech Processing** (`Spr_lab1.ipynb`)
- Basic audio loading and playback  
- Waveform visualization  
- Sampling rate, duration, amplitude  
- Basic time-domain analysis  

---

### 🔹 **Lab 2 — Framing, Windowing & Basic Features** (`lab2_spr.ipynb`)
- Speech framing  
- Windowing techniques  
- Short-term energy  
- Zero-crossing rate (ZCR)  

---

### 🔹 **Lab 3 — Real-Time Speech Recognition (Streamlit App)** (`sr_lab3.py`)
A functional speech-to-text application with:
- Live microphone transcription  
- Audio file transcription  
- Online recognition (Google API)  
- Offline recognition (PocketSphinx)  

---

### 🔹 **Lab 4 — Spectral Analysis** (`Spr_lab4.ipynb`)
- FFT computation  
- Spectrogram plotting  
- Pitch and formant analysis  
- Understanding frequency-domain features  

---

### 🔹 **Lab 5 — MFCC Feature Extraction** (`SPR_lab5.ipynb`)
Implements the MFCC pipeline:
- Pre-emphasis  
- Framing & Windowing  
- Mel Filter Banks  
- LogEnergy + DCT  
- Visualizing MFCCs  

---

### 🔹 **Lab 6 — Speech Classification Basics** (`SPR_lab6.ipynb`)
- Extracting MFCCs  
- Training a simple classifier  
- Testing & evaluation  
- Accuracy/Confusion Matrix analysis  

---

### 🔹 **Lab 7 — Speech Enhancement** (`spr_lab7.ipynb`)
- Spectral subtraction  
- Noise estimation  
- Smoothing  
- Listening & visual comparison  

---

### 🔹 **Lab 8 — Mini End-to-End Speech Processing Project** (`Spr_lab8.ipynb`)
Integrates multiple concepts:
- Audio preprocessing  
- Feature extraction  
- Simple speech analysis or recognition pipeline  

---

### 🔹 **Lab 9 — Final Speech Recognition Experiment** (`spr_lab9.ipynb`)
- End-to-end feature extraction  
- Model training/testing  
- Performance metrics analysis  
- Final recognition experiment  

---

### 🔹 **Lab 10 — Speech-Based Telephone Directory (Speech Recognition)**  
(`speech_directory_lab10.py`)
A voice-interactive contact lookup system featuring:
- Automatic CSV telephone directory creation  
- Speech-based name spelling (Google Web Speech API)  
- Microphone input with text fallback  
- Lookup of name, phone, and email  
- Interactive menu-driven interface  

Demonstrates practical integration of:
- Speech recognition  
- File handling  
- Real-world voice-controlled applications
